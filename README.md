## Docker Image Build

```
$ docker build -t ttsubo/ubuntu:upgrade-juno dockerfile/ubuntu/.
$ docker build -t ttsubo/rabbitmq:upgrade-juno dockerfile/rabbitmq/.
$ docker build -t ttsubo/haproxy:upgrade-juno dockerfile/haproxy/.
```

## Run
```
$ docker-compose -f docker-compose-single.yaml up -d
```
or
```
$ docker-compose -f docker-compose-multiple.yaml up -d
```
or
```
$ docker-compose -f docker-compose-multiple-roundrobin.yaml up -d
```
